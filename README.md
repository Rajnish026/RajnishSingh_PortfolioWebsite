# Rajnish Singh - Personal Portfolio Website

## Overview
A responsive personal portfolio website built with HTML, CSS, and JavaScript. This clean, modern interface showcases professional skills, projects, and contact information. Designed for easy customization and optimal viewing across all devices, it features both light and dark mode to enhance user experience and accessibility.

## Key Features
- **Responsive Design**: Adapts seamlessly to all screen sizes (mobile, tablet, desktop)
- **Dark/Light Mode**: Toggle between themes with persistent user preference
- **Multi-page Layout**: Organized sections for About, Skills, Projects, Resume, and Contact
- **PDF Resume**: View and download functionality
- **Contact Form**: Ready-to-use interface for visitor messages
- **Project Showcase**: Display your work with descriptions and links

## Setup & Installation
1. **Local Development**:
   - Clone/download repository
   - Place in web server directory (XAMPP/MAMP/WAMP) or use VS Code Live Server
   - Access via `http://localhost/portfolio/`

2. **Deployment**:
   - GitHub Pages: Push to repository → Settings → Pages → Deploy from branch
   - Netlify: Connect repository → Deploy

## Quick Start
1. Replace placeholder content in HTML files with your information:
   - Update name, bio, and contact details
   - Add your projects to `projects.html`
   - Replace resume PDF in `assets/resume/`
   - Customize skills in `skills.html`

2. Modify theme colors in `assets/css/style.css` if desired
3. Test all pages and links before deployment
